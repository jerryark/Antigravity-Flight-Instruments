# Antigravity - Six Pack Flight Instruments ✈️

**Antigravity** es un simulador de instrumentos de vuelo (Six-Pack) desarrollado de forma nativa para Android utilizando **Jetpack Compose** y **Kotlin**. Este proyecto replica la estética y precisión de un panel clásico de Cessna 172.

## ✨ Características principales
* **Altímetro de Precisión:** Sistema de tambor rotatorio (Rolling Drum) con actualización pie por pie.
* **Seis Instrumentos Clásicos:** Airspeed, Attitude, Altimeter, Turn Coordinator, Heading Indicator y VSI.
* **Modo Noche:** Interfaz adaptada en rojo (NightRed) para evitar el deslumbramiento.
* **Alertas por Voz:** Notificaciones del sistema GPWS (Bank Angle, Altitude).
* **Licencia MIT:** Código abierto para fines educativos y recreativos.

## 🛠️ Tecnologías
* Kotlin / Jetpack Compose
* Sensores (Acelerómetro, Giroscopio, Magnetómetro)
* GPS para telemetría en tiempo real
